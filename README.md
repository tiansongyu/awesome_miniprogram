# 微商代购小程序

## 项目简介

一句话描述：基于 Taro + NestJS + React 的全栈微商代购系统

## 技术栈

- 小程序端：Taro 4 + React 18 + NutUI + Zustand
- 后台管理：React 19 + Ant Design 5 + Vite
- 后端服务：NestJS 10 + Prisma + PostgreSQL + Redis
- 测试：Jest（后端）+ Vitest（前端）

## 功能模块

### 小程序端

- 首页：搜索、分类导航、热门商品
- 商品分类：多级分类筛选、关键词搜索
- 商品详情：图片轮播、规格选择、富文本详情、加入购物车
- 购物车：数量管理、结算
- 订单：创建订单、订单列表、订单详情、取消订单
- 退款：申请退款、退款进度查看
- 物流跟踪：实时物流信息查看
- 商品评价：订单完成后评价、评分、图片评价
- 消息通知：订单状态变更通知、系统消息
- 个人中心：手机号登录、微信登录、会员等级
- 代理中心：收益统计、邀请码、下级管理
- 优惠券：领取优惠券、下单使用优惠券
- 签到积分：每日签到获取积分、积分记录
- 商品收藏：收藏/取消收藏、收藏列表
- 收货地址管理：新增、编辑、删除、设为默认地址
- 分享邀请：生成邀请海报、分享链接
- 用户注册：手机号注册 + 邀请码绑定

### 后台管理

- 商品管理：CRUD、富文本编辑、图片上传、SKU管理、多级价格
- 分类管理：树形结构、拖拽排序、分类图片上传
- 订单管理：订单列表、状态流转、物流跟踪
- 退款管理：退款申请审核、批准、拒绝
- 商品评价管理：评价列表、审核、回复
- 消息通知系统：系统通知、订单通知、消息模板管理
- 用户管理：会员管理、代理管理、角色分配
- 优惠券系统：创建优惠券、发放管理、使用统计
- 数据概览 Dashboard：订单统计、销售额、用户增长
- 系统设置：基础配置管理
- 结算中心：代理佣金结算、收益明细

### 后端服务

- 认证：JWT + 微信登录 + 手机号登录
- 用户注册：手机号 + 邀请码注册
- 商品：多级分类（树形结构、排序、图片）、SKU、多角色价格体系
- 订单：创建、支付、发货（物流跟踪）、完成、取消
- 退款管理：退款申请、审核批准、拒绝退款
- 商品评价：评分、文字评价、图片评价
- 消息通知：系统通知、订单状态通知、消息已读管理
- 物流跟踪：物流公司管理、运单号跟踪、物流状态更新
- 优惠券：创建、领取、下单核销
- 签到积分：每日签到、积分增减记录
- 商品收藏：收藏/取消收藏
- 收货地址：CRUD、默认地址
- 用户：角色（超级管理员、代理L1/L2/L3、客户）、会员等级
- 文件上传：本地存储 + 静态文件服务
- 二维码：邀请码生成
- 分享邀请：邀请链接、邀请绑定
- 结算中心：多级代理佣金计算与结算

## 价格体系

- AGENT_L1/L2/L3：三级代理价
- MEMBER_GOLD/SILVER/BRONZE：三级会员价
- RETAIL：零售价

## 快速开始

### 环境要求

- Node.js 18+
- pnpm 9+
- PostgreSQL 16
- Redis 7

### 安装和运行

```bash
pnpm install

# 启动数据库（Docker）
docker compose -f docker/docker-compose.yml up -d

# 后端
cd apps/server && npx prisma migrate dev && pnpm dev

# 后台管理
cd apps/admin-web && pnpm dev

# 小程序
cd apps/mini-program && pnpm dev:weapp
```

### 运行测试

```bash
# 后端测试
cd apps/server && npx jest

# 前端测试
cd apps/admin-web && npx vitest run
```
