# 小程序端实现计划

**Goal:** 实现 Taro 4 + React 微信小程序客户端，包含首页、分类、商品详情、购物车、订单、个人中心、代理中心。

**Tech Stack:** Taro 4, React 18, TypeScript, Zustand, NutUI (Taro 版)

---

## 任务分解（A 完成后 B/C/D/E 并发）

| Task | 内容 | 依赖 |
|------|------|------|
| A | 项目初始化 + Taro 配置 + 请求封装 + 状态管理 + TabBar | 无 |
| B | 首页 + 分类页 + 商品详情页 | A |
| C | 购物车 + 下单流程 | A |
| D | 订单列表 + 订单详情 | A |
| E | 个人中心 + 代理中心 | A |
